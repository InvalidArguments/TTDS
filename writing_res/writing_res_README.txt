This folder is where all the writing resources will be, such as character 
dialog, story overview, character descriptions, etc. This will be where the 
game writers will be submitting their work.
NOTE: MOST, IF NOT ALL OF THIS FOLDER WILL BE DELETED UPON CREATION OF THE GAME.
IF YOU WISH TO KEEP SOME OF YOUR STUFF, SAVE IT ON TO YOUR COMPUTER.

Folder Guide:
  
  concept: This is where story concepts and ideas will be placed. Everyone will be
  able to submit here.
  
  character: This is where characters and character descriptions will be placed. Artists can
  submit here, but only images. Writers can submit here, both images and text
